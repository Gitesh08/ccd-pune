## Usage

Get a hint:
```bash
gccd-pune --date
Hint: Clouds part in Pune, day is a dozen!
Guess the date with: `gccd-pune DD-MM-YY` (e.g., `gccd-pune 12-06-25`)
```
Guess the date:
```bash
gccd-pune 12-06-25
Wrong guess! Hint: The month is a prime number!
Try again with: `gccd-pune DD-MM-YY`
```
Venue info:
```bash
gccd-pune --venue
```
See commands:
```bash
gccd-pune --help
```
