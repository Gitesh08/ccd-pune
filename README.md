# CCD Pune 2025 Interactive CLI
🌩️ Welcome to the Cloud Community Day Pune 2025 CLI! Explore the event with this interactive tool.

## Install
```bash
pip install ccdpune
```

## Usage

Guess the date:
```bash
ccdpune --date
```

Venue info:
```bash
ccdpune --venue
```

Event info:
```bash
ccdpune --info
```

See commands:
```bash
ccdpune --help
```
